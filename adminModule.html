Admin Module

The admin module comprises features that are accessible to users with administrative privileges. Presently, the admin module encompasses dashboard management, role management, and user management. From the Admin module, it is possible to create roles and manage user roles.


Dashboard Management

In Manage > Dashboards it is possible to assign the user groups that have access to a specific dashboard.


Role Management

In Manage > Roles new group/roles can be created by clicking "Create New Role" button.


The View option for each role displays all the users who are assigned with the particular role.

User Management

User creation can only be done using the Keycloak admin console. Once users are created, new roles or groups can be assigned to them from Manage > Users. Additionally, if needed, the added roles can also be removed.
